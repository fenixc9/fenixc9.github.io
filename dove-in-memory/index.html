我记忆中的那只鸽子

发表于 2023-12-16 更新于 2025-04-05 分类于 闪回 阅读次数： 
那时我上小学，家住一个老小区的四楼。

有一次，窗户外防盗网上落了一只鸽子，透过玻璃能看到。我不记得老爸怎么把它抓住的，我只记得最后它落在家里一个门框上歇脚时的样子。那只鸽子长什么样我已经不记得了，唯独记得它腿上绑了一个铁箍，写着：北京信鸽。

后来我写了纸条：“我叫xx，我住在xx，我非常喜欢你的信鸽”。老爸把这个纸条绑在了铁箍上。就把鸽子放走了。这件事我写了周记，好像是 ：一件有意义的事。

多年后，我离开故乡多年偶然想起这件事。
